  <name>Task 1: Create DwcComponentGallery and add exec-maven-plugin</name>
  <files>src/main/java/com/dwc/laf/DwcComponentGallery.java, pom.xml</files>
  <action>
Create `src/main/java/com/dwc/laf/DwcComponentGallery.java` -- a single main class with no framework dependencies.

**Structure:**
- `main()` wraps everything in `SwingUtilities.invokeLater()`
- Install L&F: `UIManager.setLookAndFeel(new DwcLookAndFeel())`
- Create JFrame "DWC Component Gallery", 900x800, EXIT_ON_CLOSE
- Main panel: BoxLayout Y_AXIS, EmptyBorder(24, 32, 24, 32)
- Title JLabel: "DWC Look & Feel - Component Gallery", bold 20f, LEFT_ALIGNMENT
- Subtitle JLabel: "All 8 themed components. Hover and press to see interactive states."
- Wrap main panel in JScrollPane with null border and unitIncrement=16

**8 component sections** (each as a private static method returning JPanel):
All section panels use BoxLayout Y_AXIS, setAlignmentX(LEFT_ALIGNMENT).
Use `BorderFactory.createTitledBorder()` for section titles.
Use vertical struts (24px) between sections.

1. **createButtonSection()** -- "JButton"
   - Row 1: state labels (Normal, Focused, Disabled) via GridLayout(1,N,8,0)
   - Row 2 "Default variant": JButton("Default") normal, JButton("Focused") with requestFocusInWindow(), JButton("Disabled") disabled
   - Row 3 "Primary variant": JButton("Primary") with `putClientProperty("dwc.buttonType", "primary")` normal, primary focused, primary disabled
   - IMPORTANT: Use `"dwc.buttonType"` NOT `"JButton.buttonType"` (the old DwcQuickDemo has a bug here)

2. **createTextFieldSection()** -- "JTextField"
   - Row: JTextField with placeholder via `putClientProperty("JTextField.placeholderText", "Placeholder...")`, JTextField("Filled text"), JTextField disabled
   - Use FlowLayout(LEFT, 8, 0) for the row, set column count to 12

3. **createCheckBoxSection()** -- "JCheckBox"
   - Row: Unchecked, Checked (setSelected), Disabled unchecked, Disabled checked (setSelected + setEnabled(false))
   - Use FlowLayout(LEFT, 8, 0)

4. **createRadioButtonSection()** -- "JRadioButton"
   - Row: Unselected, Selected (setSelected), Disabled (setEnabled(false))
   - All in one ButtonGroup
   - Use FlowLayout(LEFT, 8, 0)

5. **createComboBoxSection()** -- "JComboBox"
   - Row: Normal JComboBox with items ["Select...", "Apple", "Banana", "Cherry"], Disabled JComboBox
   - Use FlowLayout(LEFT, 8, 0)

6. **createLabelSection()** -- "JLabel"
   - Row: Normal JLabel("Normal label"), Disabled JLabel (setEnabled(false) shows disabled opacity)
   - Use FlowLayout(LEFT, 8, 0)

7. **createPanelSection()** -- "JPanel"
   - Two panels side by side in a FlowLayout row:
     a. Normal JPanel with JLabel("Normal Panel") inside, light gray background implicitly from L&F
     b. Card-mode JPanel: `putClientProperty("dwc.panelStyle", "card")`, add EmptyBorder(16,16,16,16), add JLabel("Card Panel") inside
   - Wrap card panel in a container with EmptyBorder(8,8,8,8) to prevent shadow clipping
   - Set preferred size on each panel (e.g., 200x80) so they are visible

8. **createTabbedPaneSection()** -- "JTabbedPane"
   - JTabbedPane with 3 tabs: "Active" (selected by default), "Normal", "Disabled" (setEnabledAt(2, false))
   - Each tab content: JPanel with JLabel describing the tab, with preferredSize so content area is visible (e.g., Dimension(400, 60))
   - Set maximum size on the tabbedpane to prevent BoxLayout stretch: `tabs.setMaximumSize(new Dimension(Integer.MAX_VALUE, tabs.getPreferredSize().height))` -- compute after adding tabs
   - setAlignmentX(LEFT_ALIGNMENT)

**Avoid:**
- No hardcoded colors (the L&F handles all theming)
- No `new Color(...)` calls anywhere in demo code
- No JavaFX or external framework imports
- Do NOT force hover/pressed states programmatically; those are interactive

**pom.xml change:**
Add exec-maven-plugin inside `<plugins>`:
```xml
<plugin>
    <groupId>org.codehaus.mojo</groupId>
    <artifactId>exec-maven-plugin</artifactId>
    <version>3.6.3</version>
    <configuration>
        <mainClass>com.dwc.laf.DwcComponentGallery</mainClass>
    </configuration>
</plugin>
```
